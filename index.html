Sets


In math, a set means something of a container which contains different elements inside it. They can be drawn as Venn diagrams, for example:

Here \(A\) and \(B\) are different sets (circles ) containing different elements (the numbers). Namely \(A\) contains \(0, 1, 3, 5\), and \(B\) contains \(0, 2, 4, 6\). In the language of mathematics we write this as follows:

\(A = \{0, 1, 3, 5\}\)

\(B = \{0, 2, 4, 6\}\)

Since \(0\) is inside their intersection, both set contains it. We notate the intersection (\(\cap\)) of two sets as:

\(A \cap B = \{0\}\)

Similarly there is an other type of operator, called the union (\(\cup\)) where we say that we want to know which elements belong to both sets:

\(A \cup B = \{0, 1, 2, 3, 4, 5, 6\}\)

If we want to write that one element (\(a\)) is the element of set \(C\) then we could write it as:

\(C \in a\)

In mathematics we can always substitute the values (e.g. integers) of different variables (like \(A, B, C, a\)) to their place, so we said that

\(A = \{0, 1, 3, 5\}\)

\(B = \{0, 2, 4, 6\}\)

and

\(A \cap B = \{0\}\)

\(A \cup B = \{0, 1, 2, 3, 4, 5, 6\}\)

If we substitute the values of \(A\) and \(B\) to the operations we get

\(\{0, 1, 3, 5\} \cap \{0, 2, 4, 6\} = \{0\}\)

\(\{0, 1, 3, 5\} \cup \{0, 2, 4, 6\} = \{0, 1, 2, 3, 4, 5, 6\}\)

Which is still true, and means the same as it was with the variable names.


There is a type of operator which is called the relative complementer or the set substraction (\(\setminus\)), which looks like this

\(A \setminus B = \{1, 3, 5\}\)

If we put it in words it means: "Everything which is inside \(A\) but not inside \(B\)".
